# Agile Documentation

This folder contains Agile project management artifacts used during development.

## Contents:
- Product Backlog – Defines user stories and sprint planning
- Sprint Backlog – Task-level execution details with timelines
- Standup Logs – Daily issues and resolutions
- Retrospective – Sprint-wise improvements and learnings

## Note:
The documentation was maintained and later refined using GitHub commits, team discussions, and task tracking records to ensure consistency and clarity.
